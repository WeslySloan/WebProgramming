HTML에서 < table > 설정 후 cloneNode()를 적용한 < tr > 복제

  • 복제 대상 선택 ("thread tr")
  • 복제 내용을 append할 parent 노드 선택("tbody")
  • cloneNode()를 적용하여 내용을 복제
  • 복제된 내용을 "td" 단위로 구분 : querySelectorAll("td")
  • 각 단위별로 내용을 변경하거나, sort 한 내용을 적용
  • 복제된 내용을 붙임 : appendChild(clone)
  • 기타 : innerText, textContent, innerHTML의 차이를 인터넷(stackoverflow)에서 검색해 볼 것
