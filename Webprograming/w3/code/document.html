The URL Property

    DOM 객체

  • HTML 문서는 웹 브라우저에 의해 JS의 객체로 변환된다.
  • HTML의 태그와 JS의 객체 모두 tree 구조를 표현하며, 이 둘은 1-1 대응 관계이다
  • 변환된 것은 JS 객체이므로, 이들은 JS 프로프그램에 의해서 처리될 수 있다
  • 변환된 HTML 객체 root의 변수 이름은 document 이다. 따라서 HTML은 document.xxx 형태로 접근한다
  • document 객체는 웹브라우저의 window 객체의 한 프로퍼티이다. 따라서 window.document.xxx 형태로도 접근할 수도 있다

    프로그램 설명

  • HTML이 load될 때 태그의 내용들이 화면에 display된다.
  • h1과 h2의 내용이 display되며, p는 내용이 없는 상태이다
  • 각 DOM 객체는 화면에 display되는 내용을 innerHTML 프로퍼티에 저장한다.
  • script 내의 js 코드가 실행되면서, demo (p 태그의 위치) 의 innerHTML에 url 값을 할당한다
  • 이렇게 DOM 객체의 내용이 수정되면 웹 브라우저는 이 내용을 re-display한다
  • window.document.URL 의 값은 환경에 따라 다르게 표현될 수 있다
