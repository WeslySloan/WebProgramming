DOM 노드 추가 및 삭제 : window.onload 및 addEventListener 적용

  • HTML Tag에서는 id 및 핸들러에 대한 아무런 표현을 하지 않음
  • 핸들러에서 getElementsByTagName 적용
  • load 이벤트를 script 내에서 window.onload = function () { ... } 로 표현
  • onload에서 button의 click 이벤트 핸들러를 addEventListener 로 표현
  • DOM 노드 검색: document.getElementsByTagName('tag')[index];
  • DOM 노드 생성: document.createElement('tag');
  • DOM 노드 연결: parentNode.appendChild(newNode);
  • DOM 노드 제거: nodeName.remove();
