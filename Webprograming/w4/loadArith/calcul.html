계산기 프로그램 : Operator 선택하기(select-option)

  • 입력 태그 포맷 : < input id="txt-x" type="text" value="0" dir="rtl" > 로 수를 받아 들임
  • 입력된 내용은 스트링이므로, 이를 정수로 변환하는 parseInt 함수를 적용함
  • = 버튼을 클릭하면 입력된 두 수와 선택된 연산자에 따른 연산 결과를 마지막 input 태그에 보이게 함
  • 네 개의 연산자:  +,  -,  *,  중 하나를 받아들이도록 HTML 태그 select-option 적용
  • 강의 자료 9-19 및 select-option 관련 자료 참조
  • 이벤트 핸들러에서 switch 문 적용
